EBTS 10.0.8 IEPD

This Information Exchange Package contains updates in accordance with the third quarterly EBTS Technical and Operational Update (TOU), version 10.0.8, which contains only normative changes to fbi_ebts.xsd. Please see the change log for details.

The purpose of this TOU is to document all functional changes to the most recent version of the EBTS, as an interim solution until the release of the next major version, to enable FBI/CJIS system user compliance. Please see the current TOU (below) for a comprehensive list of functional changes. Normative changes to the IEPD necessitated by this TOU are located in the Changelog, followed by all normative changes documented by previous TOUs, in reverse chronological order. The informative sample instance files have also been updated according to the TOU.

Please note that the functional changes documented by this TOU are cumulative such that user compliance requires adherence to the most recent version of the EBTS and incorporation of all functional changes documented by this and all prior TOUs since the release of the most recent version of the EBTS.
